Linux Commands: The Complete Guide

Master Linux with this comprehensive guide covering 100+ essential Linux commands with detailed examples. From basic file operations to advanced system administration, this guide will transform your Linux skills.

Linux Commands

Complete Reference Guide

Linux commands are the foundation of system administration, development, and DevOps workflows. Whether you're a beginner learning the basics or an experienced professional looking for a comprehensive reference, this guide covers over 100 essential Linux commands with practical examples.

From basic file operations to advanced system administration, network management, and process control, this comprehensive guide will help you master the Linux command line and become more productive in your daily work.

Essential File and Directory Commands

ls - List Directory Contents

Display files and directories in the current or specified directory.

Examples:

# Basic listing
ls

# Detailed listing with permissions, size, date
ls -l

# Show hidden files (starting with .)
ls -a

# Human-readable file sizes
ls -lh

# Sort by modification time (newest first)
ls -lt

# Recursive listing of subdirectories
ls -R

# List with file types indicated
ls -F

# Long format with human-readable sizes and show hidden files
ls -lah

cd - Change Directory

Navigate between directories in the filesystem.

Examples:

# Go to home directory
cd ~
cd

# Go to parent directory
cd ..

# Go to root directory
cd /

# Go to previous directory
cd -

# Go to specific directory
cd /var/log

# Go to subdirectory
cd Documents/Projects

pwd - Print Working Directory

Display the current directory path.

Examples:

# Show current directory
pwd

# Output: /home/username/Documents

mkdir - Make Directory

Create new directories.

Examples:

# Create single directory
mkdir new_folder

# Create multiple directories
mkdir folder1 folder2 folder3

# Create directory with parent directories
mkdir -p path/to/new/directory

# Create directory with specific permissions
mkdir -m 755 public_folder

# Create directory structure
mkdir -p project/{src,tests,docs}

rmdir - Remove Directory

Remove empty directories.

Examples:

# Remove empty directory
rmdir empty_folder

# Remove multiple empty directories
rmdir dir1 dir2 dir3

# Remove directory and parent directories if empty
rmdir -p path/to/empty/directory

rm - Remove Files/Directories

Delete files and directories.

Examples:

# Remove file
rm filename.txt

# Remove multiple files
rm file1.txt file2.txt file3.txt

# Remove directory and all contents (recursive)
rm -r directory_name

# Force removal without confirmation
rm -f filename.txt

# Interactive removal (ask before each file)
rm -i *.txt

# Remove directory recursively with force
rm -rf directory_name

# Remove files with specific pattern
rm *.log

cp - Copy Files/Directories

Copy files and directories.

Examples:

# Copy file
cp source.txt destination.txt

# Copy file to directory
cp file.txt /path/to/directory/

# Copy directory recursively
cp -r source_dir destination_dir

# Preserve file attributes
cp -p source.txt destination.txt

# Interactive copy (ask before overwriting)
cp -i source.txt destination.txt

# Copy with verbose output
cp -v source.txt destination.txt

# Copy multiple files to directory
cp file1.txt file2.txt /destination/directory/

mv - Move/Rename Files

Move or rename files and directories.

Examples:

# Rename file
mv old_name.txt new_name.txt

# Move file to directory
mv file.txt /destination/directory/

# Move directory
mv source_dir /destination/path/

# Move multiple files
mv file1.txt file2.txt /destination/

# Interactive move (ask before overwriting)
mv -i source.txt destination.txt

# Move with verbose output
mv -v source.txt destination.txt

touch - Create/Update File Timestamps

Create empty files or update file timestamps.

Examples:

# Create empty file
touch new_file.txt

# Create multiple files
touch file1.txt file2.txt file3.txt

# Update file timestamp to current time
touch existing_file.txt

# Set specific timestamp
touch -t 202310091200 file.txt

# Create file with specific date
touch -d "2023-10-09" file.txt

File Viewing and Editing Commands

cat - Display File Contents

Display the entire contents of a file.

Examples:

# Display file contents
cat filename.txt

# Display multiple files
cat file1.txt file2.txt

# Number all lines
cat -n filename.txt

# Show non-printing characters
cat -v filename.txt

# Display with line endings visible
cat -E filename.txt

# Concatenate files
cat file1.txt file2.txt > combined.txt

less - View File Contents (Pager)

View file contents with scrolling and search capabilities.

Examples:

# View file with less
less filename.txt

# View with line numbers
less -N filename.txt

# Search for text (press / then type search term)
less filename.txt

# Navigate: Space (next page), b (previous page), q (quit)

head - Display First Lines

Show the first few lines of a file.

Examples:

# Show first 10 lines (default)
head filename.txt

# Show first 20 lines
head -n 20 filename.txt

# Show first 5 lines
head -5 filename.txt

# Show first 1KB of file
head -c 1024 filename.txt

tail - Display Last Lines

Show the last few lines of a file.

Examples:

# Show last 10 lines (default)
tail filename.txt

# Show last 20 lines
tail -n 20 filename.txt

# Follow file changes in real-time
tail -f logfile.txt

# Show last 5 lines
tail -5 filename.txt

# Follow with line numbers
tail -fn 20 logfile.txt

grep - Search Text in Files

Search for patterns in files or text streams.

Examples:

# Search for text in file
grep "search_term" filename.txt

# Case-insensitive search
grep -i "search_term" filename.txt

# Show line numbers
grep -n "search_term" filename.txt

# Search in multiple files
grep "error" *.log

# Search recursively in directories
grep -r "function_name" /path/to/code/

# Show context lines (before and after match)
grep -C 3 "search_term" filename.txt

# Invert match (show lines that don't contain pattern)
grep -v "exclude_term" filename.txt

# Use regular expressions
grep -E "^[0-9]+" filename.txt

find - Find Files and Directories

Search for files and directories based on various criteria.

Examples:

# Find files by name
find /path -name "*.txt"

# Find files by type
find /path -type f -name "*.log"

# Find directories
find /path -type d -name "project*"

# Find files by size
find /path -size +100M

# Find files by modification time
find /path -mtime -7  # Modified in last 7 days

# Find files by permissions
find /path -perm 755

# Execute command on found files
find /path -name "*.tmp" -delete

# Find files and show details
find /path -name "*.txt" -ls

locate - Find Files by Name

Quickly find files by name using a database.

Examples:

# Find files by name
locate filename.txt

# Case-insensitive search
locate -i "filename"

# Limit number of results
locate -l 10 "*.log"

# Update locate database
sudo updatedb

File Permissions and Ownership

chmod - Change File Permissions

Modify file and directory permissions.

Examples:

# Give read, write, execute to owner
chmod 755 filename.txt

# Give read and write to owner, read to group and others
chmod 644 filename.txt

# Make file executable
chmod +x script.sh

# Remove execute permission
chmod -x script.sh

# Give write permission to group
chmod g+w filename.txt

# Remove read permission from others
chmod o-r filename.txt

# Recursive permission change
chmod -R 755 directory/

# Symbolic permissions
chmod u=rwx,g=rx,o=r filename.txt

chown - Change File Ownership

Change the owner and group of files.

Examples:

# Change owner
sudo chown newuser filename.txt

# Change owner and group
sudo chown newuser:newgroup filename.txt

# Change only group
sudo chown :newgroup filename.txt

# Recursive ownership change
sudo chown -R newuser:newgroup directory/

# Change ownership of symbolic link
sudo chown -h newuser symlink

chgrp - Change Group Ownership

Change the group ownership of files.

Examples:

# Change group
sudo chgrp newgroup filename.txt

# Recursive group change
sudo chgrp -R newgroup directory/

Process Management Commands

ps - Process Status

Display information about running processes.

Examples:

# Show all processes
ps aux

# Show processes in tree format
ps auxf

# Show processes for current user
ps ux

# Show specific process
ps aux | grep nginx

# Show process tree
ps -ef --forest

# Show processes with custom format
ps -eo pid,ppid,cmd,%mem,%cpu

top - Real-time Process Monitor

Display real-time information about system processes.

Examples:

# Start top
top

# Show processes for specific user
top -u username

# Update interval (seconds)
top -d 2

# Sort by CPU usage
top -o %CPU

# Sort by memory usage
top -o %MEM

htop - Interactive Process Viewer

Enhanced version of top with better interface.

Examples:

# Start htop
htop

# Show processes for specific user
htop -u username

# Show only specific processes
htop -p 1234,5678

kill - Terminate Processes

Send signals to processes to terminate or control them.

Examples:

# Terminate process gracefully
kill 1234

# Force kill process
kill -9 1234

# Kill all processes with name
killall nginx

# Send specific signal
kill -TERM 1234

# Kill process by name
pkill nginx

# Kill processes by pattern
pkill -f "python script.py"

jobs - List Background Jobs

Display jobs running in the background.

Examples:

# List background jobs
jobs

# List jobs with process IDs
jobs -l

# Bring job to foreground
fg %1

# Send job to background
bg %1

# Start command in background
command &

nohup - Run Commands Immune to Hangups

Run commands that continue after terminal session ends.

Examples:

# Run command with nohup
nohup long_running_script.sh &

# Redirect output to file
nohup command > output.log 2>&1 &

# Run with nohup and disown
nohup command & disown

System Information Commands

uname - System Information

Display system information.

Examples:

# Show kernel name
uname

# Show all system information
uname -a

# Show kernel release
uname -r

# Show machine hardware name
uname -m

# Show processor type
uname -p

whoami - Current User

Display the current username.

Examples:

# Show current user
whoami

# Show user ID
id

# Show detailed user information
id -a

df - Disk Space Usage

Display disk space usage for filesystems.

Examples:

# Show disk usage
df

# Human-readable format
df -h

# Show inode usage
df -i

# Show specific filesystem
df -h /dev/sda1

du - Directory Usage

Display disk usage of files and directories.

Examples:

# Show directory size
du /path/to/directory

# Human-readable format
du -h /path/to/directory

# Show summary only
du -s /path/to/directory

# Show all subdirectories
du -h /path/to/directory

# Sort by size
du -h /path/to/directory | sort -hr

# Show only directories
du -h --max-depth=1 /path/to/directory

free - Memory Usage

Display memory usage information.

Examples:

# Show memory usage
free

# Human-readable format
free -h

# Show memory usage in MB
free -m

# Continuous monitoring
free -h -s 5

uptime - System Uptime

Display system uptime and load averages.

Examples:

# Show uptime
uptime

# Show uptime in a nice format
uptime -p

Network Commands

ping - Test Network Connectivity

Test network connectivity to a host.

Examples:

# Ping a host
ping google.com

# Ping with limited packets
ping -c 4 google.com

# Ping with specific interval
ping -i 2 google.com

# Ping with timeout
ping -W 5 google.com

netstat - Network Statistics

Display network connections and statistics.

Examples:

# Show all connections
netstat -a

# Show listening ports
netstat -l

# Show with process information
netstat -p

# Show TCP connections only
netstat -t

# Show UDP connections only
netstat -u

# Show with numerical addresses
netstat -n

ss - Socket Statistics

Display socket statistics (modern replacement for netstat).

Examples:

# Show all sockets
ss -a

# Show listening sockets
ss -l

# Show TCP sockets
ss -t

# Show UDP sockets
ss -u

# Show with process information
ss -p

# Show with numerical addresses
ss -n

wget - Download Files

Download files from the internet.

Examples:

# Download file
wget https://example.com/file.zip

# Download with custom filename
wget -O custom_name.zip https://example.com/file.zip

# Download in background
wget -b https://example.com/large_file.zip

# Resume interrupted download
wget -c https://example.com/file.zip

# Download with rate limiting
wget --limit-rate=200k https://example.com/file.zip

curl - Transfer Data

Transfer data to or from a server.

Examples:

# Download file
curl -O https://example.com/file.zip

# Download with custom filename
curl -o custom_name.zip https://example.com/file.zip

# Follow redirects
curl -L https://example.com

# Show headers
curl -I https://example.com

# POST data
curl -X POST -d "data=value" https://api.example.com

# Upload file
curl -F "file=@local_file.txt" https://upload.example.com

Text Processing Commands

sort - Sort Lines

Sort lines of text files.

Examples:

# Sort file
sort filename.txt

# Sort numerically
sort -n numbers.txt

# Sort in reverse order
sort -r filename.txt

# Sort by specific field
sort -k 2 filename.txt

# Sort and remove duplicates
sort -u filename.txt

# Sort with case insensitive
sort -f filename.txt

uniq - Remove Duplicate Lines

Remove or report duplicate lines.

Examples:

# Remove duplicate lines
uniq filename.txt

# Count occurrences
uniq -c filename.txt

# Show only unique lines
uniq -u filename.txt

# Show only duplicate lines
uniq -d filename.txt

# Case insensitive comparison
uniq -i filename.txt

cut - Extract Columns

Extract columns from files.

Examples:

# Extract first column
cut -f1 filename.txt

# Extract multiple columns
cut -f1,3,5 filename.txt

# Extract with custom delimiter
cut -d',' -f1,2 filename.csv

# Extract characters
cut -c1-10 filename.txt

# Extract from specific position
cut -c5- filename.txt

awk - Pattern Scanning and Processing

Powerful text processing tool.

Examples:

# Print specific field
awk '{print $1}' filename.txt

# Print multiple fields
awk '{print $1, $3}' filename.txt

# Print lines matching pattern
awk '/pattern/ {print}' filename.txt

# Print with custom separator
awk -F',' '{print $1, $2}' filename.csv

# Calculate sum of column
awk '{sum += $1} END {print sum}' numbers.txt

# Print line numbers
awk '{print NR, $0}' filename.txt

sed - Stream Editor

Stream editor for filtering and transforming text.

Examples:

# Replace text
sed 's/old/new/g' filename.txt

# Replace in file
sed -i 's/old/new/g' filename.txt

# Delete lines
sed '5d' filename.txt

# Print specific lines
sed -n '10,20p' filename.txt

# Add text to beginning of line
sed 's/^/prefix /' filename.txt

# Remove empty lines
sed '/^$/d' filename.txt

Archive and Compression Commands

tar - Archive Files

Create and extract tar archives.

Examples:

# Create archive
tar -cf archive.tar files/

# Extract archive
tar -xf archive.tar

# Create compressed archive
tar -czf archive.tar.gz files/

# Extract compressed archive
tar -xzf archive.tar.gz

# List archive contents
tar -tf archive.tar

# Extract specific files
tar -xf archive.tar file1.txt file2.txt

# Create archive with verbose output
tar -cvf archive.tar files/

gzip - Compress Files

Compress files using gzip compression.

Examples:

# Compress file
gzip filename.txt

# Decompress file
gunzip filename.txt.gz

# Keep original file
gzip -k filename.txt

# Compress with best compression
gzip -9 filename.txt

# Compress with fastest compression
gzip -1 filename.txt

zip - Create ZIP Archives

Create and extract ZIP archives.

Examples:

# Create ZIP archive
zip archive.zip files/

# Extract ZIP archive
unzip archive.zip

# Create ZIP with password
zip -P password archive.zip files/

# List ZIP contents
unzip -l archive.zip

# Extract to specific directory
unzip archive.zip -d /destination/

User and Group Management

useradd - Add User

Create new user accounts.

Examples:

# Add user
sudo useradd username

# Add user with home directory
sudo useradd -m username

# Add user with specific shell
sudo useradd -s /bin/bash username

# Add user with specific UID
sudo useradd -u 1001 username

# Add user to specific group
sudo useradd -g groupname username

usermod - Modify User

Modify existing user accounts.

Examples:

# Change user's home directory
sudo usermod -d /new/home username

# Change user's shell
sudo usermod -s /bin/zsh username

# Add user to group
sudo usermod -a -G groupname username

# Lock user account
sudo usermod -L username

# Unlock user account
sudo usermod -U username

passwd - Change Password

Change user passwords.

Examples:

# Change own password
passwd

# Change another user's password
sudo passwd username

# Lock user password
sudo passwd -l username

# Unlock user password
sudo passwd -u username

groupadd - Add Group

Create new groups.

Examples:

# Add group
sudo groupadd groupname

# Add group with specific GID
sudo groupadd -g 1001 groupname

Service Management Commands

systemctl - System Control

Control systemd services and units.

Examples:

# Start service
sudo systemctl start nginx

# Stop service
sudo systemctl stop nginx

# Restart service
sudo systemctl restart nginx

# Enable service (start on boot)
sudo systemctl enable nginx

# Disable service
sudo systemctl disable nginx

# Check service status
systemctl status nginx

# List all services
systemctl list-units

# List failed services
systemctl --failed

service - Service Management

Control services (legacy command).

Examples:

# Start service
sudo service nginx start

# Stop service
sudo service nginx stop

# Restart service
sudo service nginx restart

# Check service status
sudo service nginx status

Package Management Commands

apt - Advanced Package Tool

Package management for Debian/Ubuntu systems.

Examples:

# Update package list
sudo apt update

# Upgrade packages
sudo apt upgrade

# Install package
sudo apt install package_name

# Remove package
sudo apt remove package_name

# Search for package
apt search keyword

# Show package information
apt show package_name

# List installed packages
apt list --installed

# Clean package cache
sudo apt clean

yum - Yellowdog Updater Modified

Package management for Red Hat/CentOS systems.

Examples:

# Install package
sudo yum install package_name

# Remove package
sudo yum remove package_name

# Update packages
sudo yum update

# Search for package
yum search keyword

# Show package information
yum info package_name

# List installed packages
yum list installed

dnf - Dandified YUM

Modern package manager for Fedora/RHEL 8+.

Examples:

# Install package
sudo dnf install package_name

# Remove package
sudo dnf remove package_name

# Update packages
sudo dnf update

# Search for package
dnf search keyword

# Show package information
dnf info package_name

Advanced Commands

cron - Schedule Tasks

Schedule commands to run at specific times.

Examples:

# Edit crontab
crontab -e

# List crontab
crontab -l

# Remove crontab
crontab -r

# Cron format: minute hour day month weekday command
# Run every day at 2 AM
0 2 * * * /path/to/script.sh

# Run every 5 minutes
*/5 * * * * /path/to/script.sh

# Run every Monday at 9 AM
0 9 * * 1 /path/to/script.sh

rsync - Remote Sync

Synchronize files and directories.

Examples:

# Sync directories
rsync -av source/ destination/

# Sync to remote server
rsync -av local/ user@server:/remote/path/

# Sync with progress
rsync -av --progress source/ destination/

# Sync excluding files
rsync -av --exclude='*.tmp' source/ destination/

# Dry run (show what would be done)
rsync -av --dry-run source/ destination/

screen - Terminal Multiplexer

Run multiple terminal sessions.

Examples:

# Start new screen session
screen

# Start screen with name
screen -S session_name

# List screen sessions
screen -ls

# Attach to screen session
screen -r session_name

# Detach from screen (Ctrl+A, then D)
# Reattach to screen
screen -r

tmux - Terminal Multiplexer

Modern terminal multiplexer.

Examples:

# Start new tmux session
tmux

# Start tmux with name
tmux new-session -s session_name

# List tmux sessions
tmux list-sessions

# Attach to tmux session
tmux attach-session -t session_name

# Create new window (Ctrl+B, then C)
# Switch between windows (Ctrl+B, then number)

Conclusion

This comprehensive guide covers over 100 essential Linux commands that every system administrator, developer, and DevOps engineer should know. From basic file operations to advanced system administration, these commands form the foundation of Linux expertise.

Remember that mastery comes with practice. Start with the basic commands and gradually work your way up to the more advanced ones. Use the `man` command to get detailed information about any command, and don't hesitate to experiment in a safe environment.

Linux commands are powerful tools that can automate tasks, manage systems, and solve complex problems. With this knowledge, you'll be well-equipped to handle any Linux environment with confidence and efficiency.

Quick Reference Tips

  • Use man command to get help for any command
  • Use command --help for quick help
  • Use history to see previously used commands
  • Use !! to repeat the last command
  • Use Ctrl+C to cancel running commands
  • Use Ctrl+Z to suspend processes
  • Use Tab for command and filename completion
